========================
Running The Test Project
========================

#. Make sure you have Django 1.1 or greater and oauth2 at 1.5.163 or greater installed
#. Change directory to la_facebook/tests/fb_la_test
#. python manage.py syncdb
#. python manage.py runserver localhost:8000
#. Open a browser and point to that URL
#. Manually do the tests 

Log settings
------------

If you don't get enough info from std.out, then go into settings and change the
logging level
